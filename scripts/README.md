# Scripts

`scripts/` contains operational commands for data preparation, metadata handling, metrics, radiomics export, and repo smoke checks.

Current entrypoints:

- `python -m scripts.make_json`
- `python -m scripts.evaluate_segmentation`
- `python -m scripts.summary_metrics`
- `python -m scripts.radiomics_analysis`
- `python -m scripts.radiomics_map_shorthand`
- `python -m scripts.smoke_check`

Figure-rendering commands live under `paper_figures/`, not here.
